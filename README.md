### Database
    - PostgreSQL
        
## `How to run Booc`
### Back-End
   * change configuration of config/database.yml to your database configuration 
   * bundle install
   * rails db:create 
   * rails db:migrate
   * rails s 
### Front-End
   * npm install
   * npm start
